# GitHub 上傳說明

建議上傳到 GitHub 的檔案：

```text
index.html
README.md
SOCIAL_POST.md
GITHUB_UPLOAD.md
```

## 每個檔案的用途

- `index.html`：抽獎機本體。
- `README.md`：GitHub 首頁會顯示的完整功能介紹。
- `SOCIAL_POST.md`：可以直接貼到社群的介紹文。
- `GITHUB_UPLOAD.md`：上傳與保存說明。

## 建議 repo 名稱

```text
Yuna-Gift-lottery-machine
```

或：

```text
gift-lottery-machine
```

## 使用方式

下載後直接打開 `index.html` 就可以使用，不需要安裝套件。

## GitHub Pages

如果想讓大家直接用網址開啟：

1. 到 GitHub repo 的 `Settings`。
2. 找到 `Pages`。
3. Source 選擇 `Deploy from a branch`。
4. Branch 選 `main`，資料夾選 `/root`。
5. 儲存後等待 GitHub 產生網址。
