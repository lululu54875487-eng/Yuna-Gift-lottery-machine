# 小U0贈物抽獎機

🎁✨ 小U0贈物抽獎機是一個給社群贈物、抽籤、多人撞號抽選使用的小工具。  
只要把清單貼上去，就可以自動抽出得主，抽完還能一鍵複製結果。

## 功能介紹

### 🎰 贈物抽獎

適合多人填選不同贈物號碼。

如果某個號碼只有一個人選，就會直接給他；如果多人撞號，系統會自動抽出一位得主。

範例：

```text
璐璐：1、16、23
抽抽：20、15
芊芊：1、15
```

### 🎲 單純抽籤

如果只是想從名單中抽一個人，也可以直接貼名字。

範例：

```text
璐璐、抽抽、芊芊
```

或是：

```text
璐璐
抽抽
芊芊
```

### 👑 VIP 指定得主

可以指定某個號碼直接給某位得主，不參加隨機抽選。

範例：

```text
21：璐璐
35：抽抽
```

### ⚙️ 每人中獎數限制

可以設定每個人最多中幾樣，支援 `1～10 樣` 或 `不限`。  
如果有人已經達到上限，後面的抽選會自動排除他，讓分配更公平。

### ✅ VIP 是否佔用中獎數

可以自己選擇 VIP 指定贈物要不要算進中獎上限。

- 勾選：VIP 指定贈物也會佔用中獎數。
- 取消：VIP 指定贈物另外計算，不影響一般抽選上限。

### 🔤 全形轉半形

清單裡如果有全形數字、全形符號或輸入法格式不同，可以一鍵轉成半形，減少格式錯誤。

### 🔎 檢查疑似重複

可以檢查可能重複的項目，例如：

- 全形半形不同
- 英文大小寫不同
- 空格不同
- 輸入法符號不同

抽獎前按一下，比較不怕漏看或重複登記。

### 📋 一鍵複製結果

抽完後可以直接複製結果，方便貼到社群、留言區或紀錄表。

### ⏩ 跳過動畫

贈物很多的時候，可以直接跳過動畫快速完成抽選。

### 🧹 清空結果 / 清空全部

可以只清空抽獎結果，也可以把清單和結果全部清掉。

## 玩法

1. 貼上贈物清單或抽籤名單。
2. 有 VIP 指定就填 VIP 欄位。
3. 設定每人最多中幾樣。
4. 選擇 VIP 要不要佔用中獎數。
5. 可先按「全形轉半形」和「檢查疑似重複」。
6. 按下開始抽獎。
7. 抽完複製結果貼出來。
